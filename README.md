Emacs configuration files. For now these are in a separate repository from the rest of my dotfiles, that could change in the future - looking into GNU Stow.

# Layout

**.emacs.d**

- early-init.el - Not a lot there, just the basics
- init.el - More standard emacs configuration and the requires for all of my customizations

**.emacs.d/lisp**

- setup-{something} All of my customizations
